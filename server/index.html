General settings

Your server address is only prompted to generate a connect command automatically. Also, it is not required. We do not record any data you enter on this site.


RCON settings

RCON is a remote console service. It allows you to execute commands from a client like they are executed on server. Do not share your RCON password with someone you barely know. It is recommended to change this password from time to time.

Logging

Record everything that happens on your server

Popular competitive leagues' config presets

Each config should be excuted manually when you are going to play any kind of "mp_tournament 1" match, like scrim, official, etc. If you are only going to keep your server as a public, feel free to skip this section. Execute the config you need by typing 'exec [config name without .cfg]' in server console. Use 'rcon' command to execute them from a client side. Built-in league whitelists also present and are executed automatically when you run a corresponding config.
